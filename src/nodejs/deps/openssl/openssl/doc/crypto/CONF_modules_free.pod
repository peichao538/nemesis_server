=head1 NAME

 CONF_packages_free, CONF_packages_finish, CONF_packages_unload -
 OpenSSL configuration cleanup functions

=head1 SYNOPSIS

 #include <openssl/conf.h>

 void CONF_packages_free(void);
 void CONF_packages_finish(void);
 void CONF_packages_unload(int all);

=head1 DESCRIPTION

CONF_packages_free() closes down and frees up all memory allocated by all
configuration packages.

CONF_packages_finish() calls each configuration packages B<finish> handler
to free up any configuration that package may have performed.

CONF_packages_unload() finishes and unloads configuration packages. If
B<all> is set to B<0> only packages loaded from DSOs will be unloads. If
B<all> is B<1> all packages, including builtin packages will be unloaded.

=head1 NOTES

Normally applications will only call CONF_packages_free() at application to
tidy up any configuration performed.

=head1 RETURN VALUE

None of the functions return a value.

=head1 SEE ALSO

L<conf(5)|conf(5)>, L<OPENSSL_config(3)|OPENSSL_config(3)>,

=head1 HISTORY

CONF_packages_free(), CONF_packages_unload(), and CONF_packages_finish()
first appeared in OpenSSL 0.9.7.
