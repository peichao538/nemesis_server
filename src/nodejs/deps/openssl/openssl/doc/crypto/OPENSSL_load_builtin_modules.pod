=head1 NAME

OPENSSL_load_builtin_packages - add standard configuration packages

=head1 SYNOPSIS

 #include <openssl/conf.h>

 void OPENSSL_load_builtin_packages(void);
 void ASN1_add_oid_package(void);
 ENGINE_add_conf_package();

=head1 DESCRIPTION

The function OPENSSL_load_builtin_packages() adds all the standard OpenSSL
configuration packages to the internal list. They can then be used by the
OpenSSL configuration code.

ASN1_add_oid_package() adds just the ASN1 OBJECT package.

ENGINE_add_conf_package() adds just the ENGINE configuration package.

=head1 NOTES

If the simple configuration function OPENSSL_config() is called then 
OPENSSL_load_builtin_packages() is called automatically.

Applications which use the configuration functions directly will need to
call OPENSSL_load_builtin_packages() themselves I<before> any other 
configuration code.

Applications should call OPENSSL_load_builtin_packages() to load all
configuration packages instead of adding packages selectively: otherwise 
functionality may be missing from the application if an when new
packages are added.

=head1 RETURN VALUE

None of the functions return a value.

=head1 SEE ALSO

L<conf(3)|conf(3)>, L<OPENSSL_config(3)|OPENSSL_config(3)>

=head1 HISTORY

These functions first appeared in OpenSSL 0.9.7.
